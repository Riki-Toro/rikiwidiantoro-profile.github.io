About Me

Mahasiswa | Full Stack Web Developer

Hallo selamat datang di website portofolio saya, perkenalkan nama saya Riki Widiantoro, umur 23 tahun dan saat ini sedang berkuliah semester 8 di Universitas Muhammadiyah Ponorogo mengambil jurusan S1 Teknik Informatika.

Disini saya mempunyai beberapa keahlian atau skills di bidang IT khususnya Full Stack Web Developer. Seperti bahasa markup HTML dan CSS, Responsive Website dengan CSS Media Queries atau Grid System, bahasa pemrograman PHP, JavaScript & DOM (Document Object Model) serta beberapa framework CSS seperti Bootstrap dan Materialize, framework PHP Codeigniter serta beberapa library JavaScript seperti jQuery dan React.js. Saya juga menggunakan Version Control System dengan Git di komputer lokal dan Github sebagai penyimpanan cloud.

Selain itu saya juga mempunyai pengetahuan lain di bidang UI/UX Design dan Data Science. Beberapa sertifikasi bootcamp dan projek yang sudah saya buat bisa di lihat di bagian Portfolio Project dan Training Certificate dibawah atau di repository github dan instagram, sekian Terimakasih.
